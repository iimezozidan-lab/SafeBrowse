# 🛡️ SaveBrowse

SaveBrowse is an open-source Android application that blocks unwanted websites using Android's Local VPN API.

## Features

- Local VPN protection
- DNS Filtering
- SNI Inspection
- DoH / DoT Support
- IPv6 Protection
- Automatic Blocklist Updates
- Privacy First
- Open Source
